## Changes

- namespaces

## Installing with Composer

Install Composer in your project:

    curl -s https://getcomposer.org/installer | php

Create a composer.json file in your project root:

    {
        "require": {
            "slim/strong": "*"
        }
    }

Install via composer:

    php composer.phar install
